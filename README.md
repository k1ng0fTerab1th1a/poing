
## How to run

1. Make sure `docker compose` and `git` are installed.
2. Clone this repo: `git clone https://github.com/k1ng0fTerab1th1a/poing`.
3. Run `docker compose up`.

After this, the app should be accessible on `http://localhost:5069`.

> Note: You can set another port and database password in `.env` file (see `.env.example`)
